E-Ink Laptop

Building a minimal E-Ink computer for distraction-free productivity

May 2025 – July 2025 • Hardware • 3D Printing • Raspberry Pi

Overview

This project explores how to create a minimal, distraction-free computing device using E-Ink technology. The goal was to reduce screen fatigue and power consumption while maintaining productivity for tasks like note-taking and code editing.

Problem

Traditional screens cause fatigue during extended use and consume significant power.

Solution

Custom E-Ink laptop with optimized software for text-based workflows.

Outcome

Functional prototype with keyboard shortcuts and multi-file navigation.
