Week 05 — Hardware Planning

Date range: Feb 17 – Feb 23

1) Summary

  • ✅ Selected core components (ESP32, e-ink display, OLED)
  • ✅ Planned wiring layout and pin usage
  • ✅ Designed system architecture
  • ✅ Finalized component bill of materials

2) What I did

Hardware Selection

  • Main controller: ESP32 — Low power, sufficient processing, built-in WiFi/Bluetooth.
  • E-ink display: Waveshare e-ink display — Proven reliability, good refresh rates.
  • OLED display: Small OLED for typing feedback.

Planning

  • Wiring layout — Mapped pin usage for displays, keyboard, and peripherals.
  • System architecture — Designed ESP32 as main controller with optional Raspberry Pi for advanced features.

3) Components selected

  • ESP32 microcontroller
  • Waveshare e-ink display
  • OLED display module
  • Mechanical keyboard switches
  • LiPo battery
  • Power management circuitry
